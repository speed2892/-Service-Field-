처음 한 번만 설정하면
70명이 실시간으로 함께 사용할 수 있어요.
설정값은 이 기기에만 저장됩니다.
STEP 1 — Firebase 프로젝트 만들기

console.firebase.google.com 접속 →
프로젝트 추가 → 이름 입력 → Google Analytics 비활성화 → 만들기

STEP 2 — 웹 앱 등록 후 구성 값 복사

프로젝트 홈 → </> 웹 아이콘 → 앱 이름 입력 → 앱 등록
→ firebaseConfig 안의 값들을 아래에 입력

STEP 3 — Firestore 데이터베이스 생성

왼쪽 메뉴 → Firestore Database → 데이터베이스 만들기
→ 테스트 모드 선택 → 위치 asia-northeast3 (서울) 선택 → 완료
